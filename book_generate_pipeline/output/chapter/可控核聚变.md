导读（优化后的课程设计脉络 / 新版 Abstract）

本课程以“一月内可参与聚变装置典型工程项目”的交付为导向，把知识重排为一条可执行、可复核的**装置方案闭环**，用“给出一份可回算、可被诊断证据支持的运行/改造方案”牵引学习，而非按学科分支线性铺陈。主线按工程决策顺序组织为九个关卡：**指标与边界定义 → 位形与运行窗口选型 → 0D/1D功率-粒子收支与Q约束 → 平衡与q剖面/线圈接口 → 稳定性边界与破裂链路风险 → 约束与输运的剖面自洽迭代 → 加热/电流驱动/加料的可控性配置 → 边界功率排出与PMI窗口 → 诊断—平衡重建—控制—集成建模的证据链闭环 → 反应堆工程约束下的一致性收敛**。

为避免跨专业学习者“会推公式但不会做工程取舍”，课程采用**理论—工程双螺旋**：每章围绕一个明确的工程决策点（如q剖面目标、稳定性边界、偏滤器热负荷窗口、TBR约束等），只引入支撑该决策所需的最小模型与判据（如三乘积、Grad‑Shafranov、能量原理、Lundquist数、经验约束定标、SOL功率扩展、辐射坍塌边界、TBR估算），并同步给出工程接口与验证路径（如自由边界平衡与线圈耦合、诊断重建、沉积剖面与驱动效率、脱靶窗口、破裂缓解注入、合成诊断与跨代码对照）。

知识图谱按认知梯度分三层循环回访：  
1）**指标层**：Q、三乘积、功率/粒子/动量收支、热负荷与中子学、材料寿命、氚库存与安全。  
2）**装置层**：位形与平衡（磁面、q、磁剪切、形状自由度）、稳定性边界（理想/电阻MHD、磁岛、VDE、破裂链路）、约束与输运（经典/新经典/湍流、L‑H与台基）。  
3）**实现层**：加热与电流驱动、加料与杂质控制、边界与PMI、诊断与重建、集成建模与验证、聚变堆工程系统。

课程结束时可形成可迁移的项目工作流：围绕给定目标（Q提升、稳态运行、热负荷受限、破裂风险受控）完成需求分解，选定位形与运行窗口，建立“模型—接口—诊断证据”的闭环，并能基于重建/仿真结果进行参数回退与故障定位。
