CSS 布局技术

Flexbox 弹性布局

  • 一维布局模型，处理行或列方向的排列
  • 主轴（main axis）：元素排列的主方向
  • 交叉轴（cross axis）：垂直于主轴的方向
  • 灵活控制对齐、间距和换行
  • 适用于导航栏、卡片组等线性布局

CSS 布局技术

Grid 网格布局

  • 二维布局系统，同时控制行和列
  • 通过 grid-template-rows 定义行
  • 通过 grid-template-columns 定义列
  • 精确控制元素的跨行跨列
  • 适用于复杂页面结构和对齐要求
