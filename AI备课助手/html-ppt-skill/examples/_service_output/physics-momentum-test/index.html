动量和动量定理

什么是动量？

  • 定义：物体质量与速度的乘积
  • 公式：p = mv
  • 单位：kg·m/s
  • 矢量性：方向与速度方向相同
  • 物理意义：描述物体运动状态的物理量

冲量的概念

  • 定义：力与作用时间的乘积
  • 公式：I = Ft（恒力情况）
  • 单位：N·s 或 kg·m/s
  • 矢量性：方向与力的方向相同
  • 注意：变力情况需用平均力计算

动量和动量定理

动量定理

  • 表达式：Ft = mv₂ - mv₁
  • 或写作：I = Δp
  • 物理意义：合外力的冲量等于物体动量的变化
  • 适用范围：惯性参考系中的所有过程
  • 矢量运算：注意正负号的选取
