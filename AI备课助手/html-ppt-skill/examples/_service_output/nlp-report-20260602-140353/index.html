自然语言处理与文本挖掘

概念

  • NLP 基础任务（面向信息抽取）
  • 核心目标：把非结构化生物医学文本转成结构化信息，便于检索、证据汇总与知识组织。
  • 本节聚焦两项关键子任务：
  • 命名实体识别（NER）：识别文本中的实体边界与类型。
  • 关系抽取（RE）：判断实体对之间是否存在特定关系，并给出关系类型与方向。
  • 医学NLP应用（与药物研发相关）
  • 关系抽取：抽取“药物—靶点/疾病/通路/不良反应”等关系，为证据链整理、知识图谱构建、检索与重定位提供输入。
  • 不良反应抽取：从文献、病例、说明书、药物警戒文本中识别不良反应信号（常见扩展字段：严重程度、时间、剂量、人群）。

45分钟课堂流程与时间分配（强化互动版）

  • 发放文本（统一输入）：提供1段生物医学摘要或说明书片段（建议包含：缩写、否定/对照、至少2个实体类型）。
  • NER（先跑通再改进）：
  • 运行 spaCy/BERT 得到实体列表；
  • 要求学生把实体按类型整理成表（实体、类型、证据片段）。
  • RE（从“能输出”到“更可信”）：
  • 基础版：基于句内实体共现 + 触发词规则（如 causes/induces/inhibits/associated with）输出候选关系；
  • 提升版（可选）：加入否定词过滤（no/not/without）与方向规则（A inhibits B）。
