动量和动量定理

动量的定义

  • 定义：物体质量与速度的乘积
  • 公式：p = mv
  • 单位：kg·m/s
  • 矢量性：方向与速度方向相同
  • 状态量：描述物体运动状态

冲量的概念

  • 定义：力与作用时间的乘积
  • 公式：I = F·Δt（恒力情况）
  • 单位：N·s 或 kg·m/s
  • 矢量性：方向与力的方向相同
  • 过程量：描述力的累积效应

动量和动量定理

动量定理

  • 内容：物体所受合外力的冲量等于物体动量的变化
  • 公式：F合·Δt = Δp = mv₂ - mv₁
  • 推导：由牛顿第二定律 F=ma 结合 a=Δv/Δt
  • 适用范围：惯性参考系中的任意物体
  • 两种表达：矢量式与标量式

动量和动量定理

典型例题分析

  • 例1：质量0.5kg的小球以10m/s速度撞墙后以6m/s速度反弹
  • 求：墙对球的冲量
  • 分析：选定正方向，计算动量变化
  • 解答：I = Δp = 0.5×(-6) - 0.5×10 = -8 N·s
  • 负号表示冲量方向与初速度方向相反

动量和动量定理

例题2：接球缓冲问题

  • 问题：为什么接球时要顺势收手？
  • 分析：动量变化量相同
  • 收手增大作用时间Δt
  • 由F·Δt=Δp可知，Δt增大时F减小
  • 结论：延长作用时间可减小冲击力

动量和动量定理

知识对比

  • 动量定理 vs 动能定理
  • 动量定理：F·Δt = Δp（矢量关系）
  • 动能定理：W = ΔEk（标量关系）
  • 动量定理关注力的时间累积效应
  • 动能定理关注力的空间累积效应
