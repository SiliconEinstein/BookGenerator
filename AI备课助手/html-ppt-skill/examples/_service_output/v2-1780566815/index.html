Python 函数式编程

Lambda 表达式

  • 匿名函数：无需 def 定义，一行搞定简单逻辑
  • 语法：lambda 参数: 表达式
  • 适用场景：排序 key、回调函数、临时计算
  • 示例：square = lambda x: x ** 2

Python 函数式编程

map / filter / reduce

  • map(func, iterable)
  • 对每个元素应用函数
  • 返回惰性迭代器
  • 示例：map(str.upper, words)
  • filter(func, iterable)
  • 保留满足条件的元素
  • func 返回 True 则保留
  • 示例：filter(lambda x: x>0, nums)
  • reduce(func, iterable)
  • 累积计算为单一结果
  • 需从 functools 导入
  • 示例：reduce(lambda a,b: a*b, [1,2,3])

Python 函数式编程

列表推导式

  • 语法：[表达式 for 变量 in 可迭代对象 if 条件]
  • 比 map/filter 更 Pythonic、更易读
  • 支持嵌套循环与多重条件过滤
  • 可扩展为字典推导式 {k:v for ...} 和集合推导式 {x for ...}
  • 性能优于等价的 for 循环写法
