动量的定义

  • 动量 p = mv（质量 × 速度）
  • 矢量性质：方向与速度方向一致
  • 单位：kg·m/s
  • 物理意义：物体运动的惯性度量

动量定理

  • 表达式：Ft = Δp = mv₂ - mv₁
  • 冲量：力在时间上的累积效应
  • 定理含义：合外力的冲量等于动量的变化
  • 适用范围：单个物体或质点系

动量守恒定律

  • 条件：系统所受合外力为零（或某方向分量为零）
  • 表达式：p₁ + p₂ = p₁' + p₂'（碰撞前后总动量相等）
  • 本质：牛顿第三定律在时间上的积累效应
  • 守恒性：自然界最基本的守恒定律之一

碰撞类型

  • 弹性碰撞：动能守恒，无能量损失
  • 非弹性碰撞：动能部分损失
  • 完全非弹性碰撞：碰后粘在一起，动能损失最大
  • 所有碰撞均满足动量守恒
