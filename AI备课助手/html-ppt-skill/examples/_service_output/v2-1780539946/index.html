第3章 动量守恒定律

动量的概念

  • 定义：p = mv，质量与速度的乘积
  • 矢量性：方向与速度方向一致
  • 单位：kg·m/s
  • 物理意义：描述物体运动状态的物理量
  • 动量变化：Δp = mv₂ − mv₁

冲量与动量定理

  • 冲量定义：I = FΔt，力与时间的乘积
  • 动量定理：F·Δt = mv₂ − mv₁
  • 物理含义：合外力的冲量等于动量的变化量
  • 适用条件：恒力与变力（取平均力）
  • 矢量形式：需分方向列方程

动量守恒定律

  • 表述：系统不受外力或合外力为零时，总动量守恒
  • 公式：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
  • 适用条件：内力远大于外力即可近似成立
  • 守恒分量：某方向合外力为零时，该方向动量守恒
  • 与牛顿第三定律的内在联系

碰撞问题分析
