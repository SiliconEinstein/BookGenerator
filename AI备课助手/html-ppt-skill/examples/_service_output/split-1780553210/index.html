第3章 动量守恒定律

动量的基本概念

  • 动量定义：p = mv，是描述物体运动状态的物理量
  • 动量是矢量，方向与速度方向相同
  • 单位：kg·m/s（国际单位制）
  • 动量变化量：Δp = p′ − p，反映运动状态的改变

动量定理

  • 内容：合外力的冲量等于物体动量的变化量
  • 数学表达：F·Δt = Δp = mv′ − mv
  • 冲量 I = F·Δt，是力对时间的积累效应
  • 动量定理是矢量方程，需按分量分别列式
  • 适用于恒力与变力（变力取平均力）

动量定理的应用

  • 求解碰撞、打击、爆炸等瞬时过程问题
  • 安全气囊原理：增大 Δt 以减小作用力 F
  • 鸡蛋碰石头：力等大反向，但加速度不同
  • 跳远缓冲：屈膝落地延长作用时间减小冲击力
  • 解题步骤：选对象 → 定正方向 → 分析受力 → 列方程
