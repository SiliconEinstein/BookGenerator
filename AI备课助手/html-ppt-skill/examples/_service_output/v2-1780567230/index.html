Python 函数式编程

Lambda 表达式

  • 匿名函数的快速定义方式
  • 语法：lambda 参数: 表达式
  • 适用于简单的一次性函数
  • 常与 map/filter 配合使用
  • 示例：lambda x: x * 2
  • 多参数：lambda x, y: x + y
  • 返回值自动计算
  • 不支持多行语句

Python 函数式编程

map / filter / reduce

  • map(func, iter)
  • 对每个元素应用函数
  • 返回迭代器
  • 示例：map(lambda x: x**2, [1,2,3])
  • filter(func, iter)
  • 筛选满足条件的元素
  • 返回迭代器
  • 示例：filter(lambda x: x>0, nums)
  • reduce(func, iter)
  • 累积计算结果
  • 需导入 functools
  • 示例：reduce(lambda x,y: x+y, [1,2,3])

列表推导式

  • 更 Pythonic 的函数式写法：[表达式 for 变量 in 可迭代对象 if 条件]
  • 基础用法：[x*2 for x in range(5)] → [0,2,4,6,8]
  • 带条件筛选：[x for x in range(10) if x%2==0] → [0,2,4,6,8]
  • 嵌套推导式：[[j for j in range(3)] for i in range(2)]
  • 性能优于 map/filter，可读性更强
