动量的定义与性质

  • 定义：p = mv，质量与速度的乘积
  • 动量是矢量，方向与速度方向一致
  • 单位：kg·m/s（国际单位制）
  • 动量是状态量，描述物体在某一时刻的运动状态
  • 动量的分量形式：pₓ = mvₓ，py = mvy

动量定理

  • 表述：合外力的冲量等于物体动量的变化量
  • 公式：FΔt = mv₂ − mv₁（即 I = Δp）
  • 本质：牛顿第二定律的时间积分形式
  • 适用于恒力与变力（变力取时间平均值）
  • 分量形式：在各坐标轴方向上独立成立

动量定理与牛顿定律的关系

  • 动量定理由牛顿第二定律推导而来，两者等价
  • 牛顿定律侧重瞬时关系（F = ma）
  • 动量定理侧重过程累积（FΔt = Δp）
  • 打击、碰撞等短时间大力问题优先用动量定理
  • 动量定理为后续学习动量守恒定律奠定基础
