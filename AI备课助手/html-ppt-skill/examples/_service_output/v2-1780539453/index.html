动量的定义与物理意义

  • 定义：p = mv，质量与速度的乘积
  • 矢量性：方向与速度方向一致
  • 单位：kg·m/s
  • 物理意义：描述物体运动状态的物理量
  • 动量变化反映运动状态的改变

冲量的概念

  • 定义：I = Ft，力与作用时间的乘积
  • 矢量性：方向与力的方向一致
  • 单位：N·s，与 kg·m/s 等价
  • 变力冲量：I = ∫F dt（积分形式）
  • 冲量是力对时间的积累效果

动量定理的推导

  • 出发点：牛顿第二定律 F = ma
  • 变形：F = m(Δv/Δt) → FΔt = mΔv
  • 一般形式：I = Δp = p₂ − p₁
  • 含义：合外力的冲量等于动量的变化量
  • 既适用于恒力，也适用于变力（积分形式）

易错点与注意事项

  • 动量是矢量，运算时注意方向
  • 动量变化量 Δp = p末 − p初，不是大小相减
  • 冲量由合外力计算，不要遗漏力
  • 区分动量定理（过程量）与牛顿定律（瞬时量）
  • 单位换算：1 N·s = 1 kg·m/s
