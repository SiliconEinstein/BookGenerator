# AI备课助手第一阶段任务拆解

## 1. 目标

第一阶段目标是完成“可开发、可联调、可迭代”的项目基础设计，不追求一次性把全部功能做完，而是优先明确协议、架构与最小闭环。

---

## 2. 第一阶段工作包

### 工作包 A：需求与协议定稿
- 明确课程目录 JSON 输出结构
- 明确章节大纲 JSON 输出结构
- 明确 PPT 中间描述 JSON 输出结构
- 定义上传方式、任务接口、状态接口、结果接口、错误码

### 工作包 B：技术架构设计
- API 层
- Agent 编排层
- 业务模块层
- 存储层
- 日志与观测层

### 工作包 C：开发骨架设计
建议至少拆分：
- `src/api`
- `src/modules/course-outline`
- `src/modules/chapter-outline`
- `src/modules/ppt`
- `src/modules/task`
- `src/modules/llm`
- `src/shared/schema`
- `src/shared/storage`
- `src/shared/logger`

### 工作包 D：最小闭环设计
- 上传 PDF
- 生成课程目录 JSON
- 生成单章节大纲
- 生成基础 PPT

---

## 3. 建议优先级

### P0：必须先做
- 三类 JSON Schema 定稿
- API 契约定稿
- 任务机制设计
- Agent 流程设计

### P1：第一批开发
- 课程目录解析链路
- 单章节大纲生成链路
- 基础任务查询接口

### P2：第二批开发
- 批量章节生成
- PPT 页面中间描述
- PPT 模板抽象

### P3：后续增强
- OCR 兜底
- 多学科 Prompt 模板
- 多模板 PPT 风格
- 质量评估与人工修订闭环
